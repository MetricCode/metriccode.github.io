The History of Computer Operating Systems(OS)
Where it all began...

From a simple IBM project to youe everyday common tool!


The first computers did not have operating systems as each program running on these computers had to include all the code needed to run on the computer, communicate with the connected hardware and perform the calculation intended by the progam.

Thus this necessity led to the owners of the central computer to come up with system software that facilitated the writing and execution of programs hence the first operating systems were born.

NB. these operating systems had no GUI.(Graphical Interface) ...

IBM was the first computer manufacturer to distribute computers whuchhad an operating system. The first operating system created to run a single IBM computer was created by the General Motors Company, The input/output system of Genral Motors and North American Aviation (GM-NAA I/O) and ran in the IBM 704 Computer in 1956.

Later in the following decades,after the development of the first OS which could run a program at a time, computers began to include more and more software programs, sometimes called libraries, whichcame together to create the start of today's operating system. In the late 1960's, the first version of the Unix Os was made. It was created using C language and was relatively free during its first years. Unix quickly adapted to the new systems and was wide accepted.

Many modern operating systems date back to or rely on the Unix OS. Microsft Windows was developed in response to an IBM request for an OS to run its range of PC's. (Personal Computers). Windows first operating system was MS-DOS and was built in 1981 after it was bought from Seattle Company Programs and modified it to meet IBM's requirements. The computer was known as Acorn. Windows name first came into use in 1985 when a graphical User Interface (GUI) was created and merged with MS-DOS.
